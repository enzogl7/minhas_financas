# Minhas finanças
Aplicação desenvolvida em Python com Tkinter para o controle e análise de finanças pessoais.
<br>
O projeto foi criado como trabalho final da disciplina de Desenvolvimento Rápido de Aplicações em Python, com o objetivo de integrar conceitos de interfaces gráficas, persistência de dados e visualização de informações.
<br>
A aplicação permite o registro de receitas e despesas, o armazenamento das transações em arquivo CSV e a geração de gráficos interativos que ajudam o usuário a visualizar sua situação financeira de forma simples e intuitiva.
